智能农业大棚

发表于2023-03-19|更新于2023-03-20
|阅读量:

项目描述：
设计一个智能农业系统，要求如下：
（1）以 TFT 屏作为显示终端，显示大棚温湿度（单位摄氏度、相对湿度）、 光照（单位 lux）信息；显示补光灯、鼓风机（电机模拟）工作状态（开启或关 闭）。其中大棚温湿度来自于温湿度传感器、光照来自于光敏电阻或其他传感器、补 光灯为高亮 LED 输出。
（2）温湿度数据、光照数据每隔 2 秒动态刷新；采用合理方式调整温湿度、 光照信号变化，以产生可见动态效果。
（3）当大棚温湿度高于某限定值，语音模块报警（报警内容：温湿度超 限），同时鼓风机（电机模拟）工作 5 秒并关闭；当光照低于某限定值，语音模块报 警（报警内容：光照亮度偏低），补光灯亮起，高于某限定值，补光灯灭。
（4）当补光灯亮起后，具有自动调节亮度功能，分为三档“弱、中、强”，根 据不同光照强度切换对应档位。
（5）设计 4x4 矩阵键盘输入，设置室温、湿、光照限定值报警功能，并实现 声、光报警。
（6）设计 RFID 射频 IC 卡功能，可以设置卡的级别：管理卡，用户卡。刷 “管理卡”，屏幕显示管理卡，同时可以制作用户卡；刷“用户卡”，屏幕显示用户 卡。
（7）刷“用户卡”，按下功能按键，实现历史数据查询功能，可以记录 30 秒数 据（共 15 次），通过翻页等可实现数据历史查询。

工作原理设想：
显示：通过TFTLCD屏进行数据以及的显示。
补光灯：通过PWM输出来让LED灯充当补光灯，根据不同的光照强度的环境调节LED的发光程度。
鼓风机：使用直流电机来模拟鼓风机的运行。
温湿度：本设计中的温湿度通过DHT11来获取。
光照：光照强度通过光敏电阻获得的不同值来表示。
刷新时间2s:通过TIM2通用定时器来定时获取数据输出。
报警：不会用语音模块，所以退而求其次用了蜂鸣器。
键盘输入：键盘使用矩阵键盘扫描方式来实行中断。
RFID卡管理：RC522通过识别不同的卡片ID号来识别不同的卡，主观上认定“管理卡”和“用户卡”并对其识别做出判断和输出。
历史数据：历史数据通过flash存储，实现断电可保存数据。
翻页：通过LCD的屏幕刷新和按键中断的结合实现翻页。

main.c展示如下
使用板子型号为STM32F103ZET6
大部分模块使用杜邦线接在引脚上使用
涉及使用RFID技术模拟的权限卡模式，PWM，IIC，USART,EEPROM等
主要传感器：
温湿度传感器，光敏传感器，TFTLCD，矩阵键盘，蜂鸣器。

        //使用按键:PB1:蜂鸣器  PB2:电机 推挽输出，10MHz
        void ShuChu_Init(void){
            RCC->APB2ENR|=1ul<<3;
            GPIOB->CRL|=0X00000110;
            GPIOB->CRL&=0XFFFFF11F;
            GPIOB->ODR|=(1UL<<1);
            GPIOB->ODR&=~(1UL<<2);
        }

        /*每隔两秒采集传感器数据*/
        void TIM2_IRQHandler(){
            static int i=0;
            NVIC->ICPR[0]|=1ul<<28;
            TIM2->SR&=~(1ul<<0);
            i++;
            if(i==2){
                i=0;
                date=Get_pcf8591();  //采集模拟数据：数据来自于pcf8591t模数转换模块
                //采集实际传感器数据
                while(DHT11_Read_Data(&Wendu,&Shidu))	//读取温湿度值)
                {
                    DHT11_Read_Data(&Wendu,&Shidu);	//读取温湿度值
                }
                storeto_FLASH(FLASH_SAVE_ADDR,Wendu);	
                storeto_FLASH(FLASH_SAVE_ADDR,Shidu);
                    storeto_FLASH(FLASH_SAVE_ADDR,date.Guangmin);
                POINT_COLOR = BLUE;
                if(!ALREADY)
                {
                    LCD_ShowNum(30+90,170,Wendu,2,16);	//显示温度	   		   
                    LCD_ShowNum(30+40,190,Shidu,2,16);		//显示湿度
                    LCD_ShowNum(30+40,210,date.Guangmin,3,16);		//显示光照
                }
            }
        }

        void EXTI0_IRQHandler(){
            delay_ms(50);
            EXTI->PR|=1UL<<0;
            NVIC->ICPR[0]|=1UL<<6;
            GPIOC->CRL=0x88881111;
            GPIOC->ODR&=~((1ul<<4)|(1ul<<5)|(1ul<<6)|(1ul<<7));
            GPIOC->ODR|=0xffffff0f;
            if(acess==1 &&(GPIOC->IDR&(1<<4))){
                //按键S1
                dead_Wendu+=1;//温度限定值加一
                k++;
            }
            else if(acess==1 &&(GPIOC->IDR&(1<<5))){
                //S5
                dead_Guangzhao+=10;//光照+10

            else if((GPIOC->IDR&(1<<6))){
                //S9
            }
            else if((GPIOC->IDR&(1<<7))){
                //s13
                FLASH_Unlock();
                FLASH_ErasePage(FLASH_SAVE_ADDR);
                FLASH_Lock();//清空扇区

            else
            {;}

            GPIOC->ODR|=(1ul<<0)|(1ul<<1)|(1ul<<2)|(1ul<<3);
            GPIOC->ODR&=0xffffff0f;
            delay_ms(100);
        }
